# Turnspike Dotfiles

turnspike's nix settings. Sane defaults and tweaks for:

  * zsh
  * spacemacs
  * byobu

# Installation

From the project directory, run:

    > chmod u+x install.sh
    > ./install.sh

Note - this will also install [homeshick](https://github.com/andsens/homeshick) if needed.
